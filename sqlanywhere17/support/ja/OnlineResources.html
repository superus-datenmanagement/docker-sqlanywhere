SQL Anywhere の製品とライセンス

SQL Anywhere は、いくつかの基本パッケージと、追加の展開ライセンスおよび別途ライセンスされるコンポーネントという形で提供されます。まず、お客様の要件に最もよく合う基本パッケージを選択します。基本パッケージには、(a) サーバ/シート、(b) プロセッサベースの 2 種類のライセンスモデルがあります。

サーバ/シートモデルの場合、SQL Anywhere ネットワークサーバとそれぞれの数のユーザライセンスが含まれています。各ユーザはネットワークシートとしてライセンスされ、SQL Anywhere サーバにクライアントアクセスできます。また各ユーザには、同期および同期付き Ultra Light を含む SQL Anywhere パーソナルサーバもライセンスされます。

代わりに、SQL Anywhere サーバ CPU ライセンスを選ぶこともできます。このパッケージでは、1 台のマシン上の SQL Anywhere ネットワークサーバが、購入した CPU ライセンス数だけライセンスされます。サーバにネットワークシートとして接続できるユーザの数に制限はなく、より大規模なクライアント/サーバ型の展開や Web ベースの展開、またはユーザやシートを数えるよりプロセッサベースのライセンスが好ましい場合に最適です。

展開ライセンス

ユーザ数を増やしたりコンポーネントを追加したりする必要がある場合でも、問題ありません。ネットワークシート (クライアント)、ネットワークサーバ、パーソナルサーバ、同期、Ultra Light には、追加ライセンスのご用意があります。

開発者サポート

オンラインサポート
SAP では、お客様が世界レベルのソリューションを迅速かつコスト効率よく開発および展開できるよう、正確でタイムリーな技術支援を提供します。このサポート提供には、お客様のビジネスニーズに合わせてさまざまなレベルがあります。標準の年間プランでは、製品の更新が提供され、お客様の問い合わせ担当者を複数登録でき、案件数に制限がありません。プレミアムプランでは、これに加えて、時間延長、対応まで時間の短縮、Direct-to-Expert サポートが含まれます。

SQL Anywhere Forum
このフォーラムは SQL Anywhere のカスタマサービス & サポートがモニタリングしており、開発に関する問題について開発者コミュニティと議論するのに便利なリソースです。

DocCommentXchange
SQL Anywhere のマニュアルを表示してコメントしたり、他のユーザによるコメントを表示したりできます。

SQL Anywhere パートナープログラム
SQL Anywhere パートナープログラムは、業界をリードするテクノロジに基づく付加価値サービスやプログラムをメンバーの皆様にお届けすることを目的としています。SQL Anywhere パートナープログラムでは、販売およびマーケティングのプログラムやサービスを提供して、メンバーの皆様が今日の市場で競争力を発揮して勝ち抜けるようお手伝いします。

SAP プロフェッショナルサービス
SAP プロフェッショナルサービスは、SQL Anywhere プロジェクトの設計、開発、展開中に次のようなサービスをお客様に合わせて提供します。

  • SQL Anywhere の最新バージョンへのアップグレードの計画と実施
  • SQL Anywhere アーキテクチャの理解に不可欠な知識移転
  • 環境のパフォーマンスと全体的な品質の保証
  • Mobile Link 同期ソリューションの設計および開発の支援
  • SQL Anywhere 高可用性機能の設定と実装

トレーニングと認証
SAP では、SQL Anywhere トレーニングをご用意してアプリケーションの開発から展開までを支援します。初級/上級に関係なく、標準の教育コースとカスタマイズされたトレーニングプログラムがあり、データベースの機能、同期、ハンドヘルド展開などのトピックをカバーしています。
