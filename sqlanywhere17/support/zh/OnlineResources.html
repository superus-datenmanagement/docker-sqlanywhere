SQL Anywhere 产品和授权

SQL Anywhere 以具有额外部署许可和单独授权组件的多个基础软件包形式提供。首先，选择最能满足您需求的基础软件包。基础软件包以两种授权模式提供：(a) 服务器和用户存取设备模式；(b) 基于内核的模式。

服务器和用户存取设备模式包括 SQL Anywhere Network Server 以及所述数量的用户许可证。其中每个用户都被授权为联网客户端存取设备，以向 SQL Anywhere Server 提供客户端访问。每个用户还被授权使用远程数据库客户端存取设备，其中包括 SQL Anywhere Personal Server with Synchronization 和 Ultralite with Synchronization。

客户也可选择 SQL Anywhere Server 基于内核的许可。此软件包授权 SQL Anywhere Network Server 在一台计算机上具有所购买数量的内核许可。对可作为联网客户端存取设备连接到服务器的用户数量没有限制，这就使此软件包特别适合于较大型客户端/服务器和基于 Web 的部署，或者客户偏爱基于内核的授权模式而不想要计数用户或存取设备的情况。

部署许可

需要授权更多的用户或额外的组件吗？没问题。可针对额外的网络存取设备（客户端）、网络服务器、个人服务器、同步和 Ultralite 进行授权。
